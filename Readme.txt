Color Flipper

Описание

Color Flipper — это веб-приложение, которое позволяет случайным образом менять цвет фона страницы при каждом нажатии кнопки. Этот проект был создан для практики работы с JavaScript и CSS, с фокусом на динамическое изменение стилей страницы.

Установка
	1.	Склонируйте репозиторий: git clone <ссылка на репозиторий>
    2.	Перейдите в директорию проекта: cd color-flipper
    3.	Откройте файл index.html в браузере, чтобы начать использовать приложение.

Использование
	1.	Откройте файл index.html в вашем браузере.
	2.	Нажимайте на кнопку “Change Color” для смены фона страницы на случайный цвет.

Тесты

Тесты не предусмотрены в текущей версии проекта.

Стек технологий
	•	HTML — структура веб-страницы.
	•	CSS — оформление и стилизация страницы.
	•	JavaScript — логика изменения фона.

Контрибьютинг
	1.	Форкните репозиторий.
	2.	Сделайте необходимые изменения.
	3.	Создайте Pull Request для внесения изменений.
